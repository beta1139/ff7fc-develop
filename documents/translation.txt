■ モンスター名・アイテム名英訳方針 ■

翻訳係: ff7fc.translate@gmail.com


モンスター名・アイテム名など以下の方針で英訳しました。異論は認めます。
いいアイディアがあったらどんどん修正してください。
基本的には公式(?)の英語名をベースにしています。

参考：
http://www11.plala.or.jp/tonberry/
http://finalfantasy.wikia.com/wiki/
など



■ モンスター名（最大10文字）

○ 1単語で10文字以内のものはそのまま使用する
例）
	Sweeper
	Grunt
	など

○ 複数の単語で構成されるもの
初めから 10 文字に収まるものとスペースを除けば10文字に収まるものがあるが、
たとえば "Air Buster" のようにそのままで使えるものと "GuardHound" のように
スペースを除いたものが混在すると全体の統一感がとれないため、
複数の単語で構成されるものは全てスペースを除いた形を用いる。
例）
	Guard Hound -> GuardHound
	Air Buster  -> AirBuster
	など

○ 複数の単語でスペースを除いても10文字を超えるものは、形容詞的・補助的な
ほうの単語を１文字のイニシャルにして "." でつなげる。
例）
	Missile Launcher  -> M.Launcher
	Custom Sweeper    -> C.Sweeper
	Jenova LIFE       -> Jenova.L
	など

○ １単語で10文字を超えるものおよび複数の単語で上記の短縮を行っても10文字に
収まらないものについてはケースバイケースで短縮を工夫する。
	Spedial Combatant -> S.Combat
	Proto Machinegun  -> ProtoM-Gun

FFオリジナルの名称で１単語が長いものなどは無理やりぶっちぎるしかないものも。
	Nerosuferoth      -> Nerosufero
	Christopher       -> Christoph

あるいは単語っぽいところで無理やり２単語とみなして片方をイニシャルにするとか。
	Lessaloploth      -> L.Loploth
	Pollensalta       -> P.Salta


（訳者個人の感覚で）公式の英語名称がどうにもおかしいと思われるものは、
よりふさわしいと思えるものに変更した。

○ 名称が一般的な単語から成り立っていて、公式が訳語としておかしいもの
例）
	警備兵       -> Guard (公式: MP)
	警備兵は Military Police ではないと思う。

	ヘクトアイズ -> HectoEyes (公式: 8 Eyes)
	グラフィックを見ると明らかに目が 8 個以上あるし…
	日本語名に準じた hecto (100) のほうがふさわしいと思う。

	アーリマン   -> Ahriman (公式: Allemagne)
	ゾロアスター教の悪神アーリマンの英語スペルを使用した。
	なんでアーリマンが Allemagne になるのかまったく意味不明。
	これはフランス語でドイツをあらわす単語だ。

	キマイラ     -> Chimera (公式: Harpy)
	グラフィックはどう見てもハーピー（鳥の怪物）には見えない。

	水球         -> WaterBall (公式: Waterpolo)
	waterpoloって、競技の水球だよ…

○ FFオリジナルではなく元ネタのあるモンスター名のスペルがおかしいもの。
公式を翻訳した人が元ネタを知らなくて適当なスペルを考えたんでしょうかね。
辞書にも載っているほどの古典的なものはやはり本来のスペルを使うべきかと。
例）
	コカトリス         -> Cockatrice (公式: Cokatolis)
	バジリスク         -> Basilisk   (公式: Bagrisk)
	クアール           -> Coeurl     (公式: Cuahl)
	インセクトキマラ   -> ChimeraBug (公式: Kimara Bug)
	マキシマムキマイラ -> MaxChimera (公式: Maximum Kimara)



■ アイテム名（最大9文字)

公式そのままで使えるものはそのまま。
例）
	ポーション       -> Potion
	ハイポーション   -> Hi-Potion
	エクスポーション -> X-Potion
	エーテル         -> Ether

複数の単語になる場合は単語間スペースを除去 (除去しないですむ場合でも)。
例）
	けむりだま       -> SmokeBomb (公式: Smoke Bomb)
	クモのいと       -> SpiderWeb (公式: Spider Web)

あとはひたすら字数削減のため省略・置き換えなど
例）
	エーテルターボ   -> X-Ether   (公式: Turbo Ether)
	しにがみのキッス -> FatalKiss (公式: Kiss of Death)
	やまびこえんまく -> EchoSmoke (公式: Echo Screen)

Drink -> Drug, Powder -> Dust, Curtain -> Veil といった似た意味で
より短い単語への置き換えとか
例）
	スピードドリンク -> SweftDrug (公式: Speed Drink)
	SpeedDrug だとあまりにもヤバいクスリっぽいので前半も変えました ^^;

	ドリームパウダー -> DreamDust (公式: Dream Powder)
	ひかりのカーテン -> LightVeil (公式: Light Curtain)

場合によってはかなりラディカルな置き換えもしました。
例）
	バジリスクのつめ -> MedusaEye (公式: Vagyrisk Claw)
	公式はそのまま「バジリスクのつめ」です（スペルがおかしいけど）。
	Basilisk だけで 8 文字になってしまうのでバジリスクは捨てて、
	「犠牲者を石にする」というところからギリシャ神話のメデューサ
	を引っ張ってきました。
	シンプルに PetroClaw/LithoClaw (≒石の爪）という案もあるな…


歴史的経緯のためか、あるいは公式翻訳者がよくわかってなかったのか、
アイテム名にはとにかく変な訳も少なくないので、こういうのも修正。
例）
	きんのはり       -> GoldPin   (公式: Soft)
	そりゃ石化を解くアイテムアイテムだけど、Soft ってのは…
	字数があれば Golden Needle にしたかったところ。

	フェニックスのお -> FenixTail (公式: Phoenix Down)
	Down はダウンジャケットとかのダウンで「羽毛」ですね。
	フェニックスの尾はやっぱり長〜い尾羽根じゃないかな。
	これも字数があれば Phoenix Tail ですね。

	もずのはやにえ   -> Tadpole   (公式: Impaler)
	公式の Impaler は「串刺し」というくらいの意味になるんですが、
	日本語の「はやにえ」のようにイメージに直接つながる言葉ではないので
	この単語を見てカエルにつながる連想をする人はいないんじゃないかと…
	Shrike（もず）も検討してみたのですが、英語圏の人にカエルを連想させる
	のはちょっと無理があると思い、ごくシンプルに「おたまじゃくし」に。



■ キーアイテム名 (最大12文字)

大部分は公式をそのまま利用できている（訳もすごく変なものはわずか）
例) 
	シルクのドレス   -> Silk Dress (公式: Silk Dress)
	ブロンドのかつら -> Blonde Wig (公式: Blonde Wig)

似た意味で短い単語に置き換えたもの
例) 
	フラワーコロン   -> Floral Scent (公式: Flower Cologne)
	くすりやクーポン -> Drug Coupon  (公式: Pharmacy coupons)

アイテム名の一部を省略したもの
例）
	みずがみさまのうろこ -> Lev. Scale (公式: Leviathan Scales)

キーアイテムの名称はストーリー上で意味をもつことが多いため、
一部を省略した場合は入手時のメッセージで "正式アイテム名(省略名)"
と表示して、どれがどのアイテムなのかが分かるようにする。
例）
	You got the "Leviathan Scale(Lev. Scale)"!


■ マテリア（最大8文字）

一単語ですむ単純なものはほぼ公式をそのまま。
例）
	Fire
	Ice
	Restore
	Heal

長いものはせめてマテリアの性質がわかる程度の名称を検討しました。
そもそもマテリアは日本語でも名前だけでは意味のわかりにくいのが多いので…

以下の２個のみ、現在のところ 9 文字のままです。
	Alexander
	Leviathan
マテリアメニューの選択ペインでは１文字右にはみ出してしまいます。
情報ペインには 9 文字分の領域があるのできちんと表示されます。

文字数の問題ではなく独断であえて変えたものもあります。
例）
	ぜんたいか      -> AimAll (公式: All)
	初心者の館で拾ったときに "You got All materia!" と表示されて
	"全てのマテリアを手に入れた" と読めてしまうのはどうかと思ったので
	誤解の余地のないような名前にした。

	てきのわざ      -> Knack (公式: Enemy Skill)
	Skill はどちらかといえば「熟練の技術」といった意味での「わざ」です。
	「特殊技能」「超能力」といった意味合いでは Knack のほうが近いかと。

	ものまね        ー＞Mimic (公式: Mime)
	この二つの単語の区別は非常に微妙なんで感覚的なところが大きいのですが
	Mime は「なにかをしている動作をまねる」というニュアンスがあります。
	要はパントマイムのマイムですね。
	これに対して Mimic は「他の人の動作・様子をまねる」です。
	FF7 のものまねは敵の動作をまねるので Mimic にしました。



■ アクセサリ（最大8文字）
■ 防具名（最大8文字）
■ 武器名（最大8文字）

この長さではまともに訳すのはほぼ無理。
基本的に武器防具類の名称は深い意味を持たないことも多いので、身につけて
いるものの区別ができればよいかと。
ただしほのお、れいき、など属性系の言葉はそのアイテムの性質にかかわってくる
ことが多いので、これはなるべく似た意味を持つ言葉に置き換えるようにする。

アクセサリはまだしも、武器・防具はけっこうどうにもならなかった orz
実際のプレイでは別途リストを用意して見ながらやってもらうのかなあ…



■ 独立マテリア・支援マテリア機能（16文字）
マテリアメニューの情報ペインでの表示に使用されるのみ？
基本的にマテリア名と同じ。
マテリア名で無理に縮めていたものは本来の内容に戻したりしている。
例）
	マテリア名：ChocLure 機能：Chocobo Lure



■ 召還魔法（最大12文字）
まほうメニュー画面で12文字。
バトル中メニュー・バトル中発動画面では最大文字数不明。
文字数制限より、一部公式の変な訳のほうが気になる。

	ひっさつわざ            DeathBlow
	デブチョコボ            Fat-Chocobo
	ダイヤモンドダスト      Diamond Dust
	じごくのかえん          Hellfire
	だいかいしょう          Tidal Wave
	せいなるしんぱん        Judgement
	メガフレア              Mega Flare
	ギガフレア              Giga Flare
	テラフレア              Tera Flare
	アルティメットエンド    Ultimate End
このあたりは公式そのままです。

	てんせいのほのお        Rebirth Fire（公式: Phoenix Flame）
	さばきのかみなり        Doom Thunder (公式: Judgement Bolt)
	あんこくのかま          Dark Caldron (公式: Black Cauldron)
長さを縮めるための置き換え

	だいちのいかり          Earth Wrath （公式: Anger of the Land）
神や大地のようなものによる怒りは wrath のほうがしっくりくるかと。

	ざんてつけん            Iron Ripper（公式: Steel Bladed Sword）
これはひどい。公式は"鋼の刃の剣"かな？「鉄をも切り裂く剣」をイメージしてみた。

	グングニルのやり        Gungnir（公式: Gunge Lance）
なんで公式でわざわざ名前を変えてあるのか？意味不明。

	トライディザスター      Tri-Disaster（公式: Tetra Disaster）
これも公式でトライ(3)からTetra(4)に名前を変えている理由が不明。
火・氷・雷の3属性攻撃なのに…

	てんちほうかい          Big Crunch（公式: Disintegration）
公式は何かが違う気がする（素粒子崩壊とかでよく使う言葉、て感じがする）。
本当はなにか神話的な世界の終わりをあらわす言葉にしたい感じなんだけど、
Typhon のギリシャ神話には終末の概念がないしなあ。



■ 敵の技（最大12文字? 多分召還魔法と同じ）
これも変なのは意外と少なかった。文字数調整のための置き換えが主。

例）
	てんしのささやき -> Angel Voice (公式: Angel Whisper)
	しのせんこく     -> Death Order (公式: Death Sentence)



■ 追加コマンド（最大？文字・バトルメニュー構成が不明）
とりあえず6文字を想定して検討中。



■ 魔法（最大4文字・バトルメニュー構成のため）
4文字でどーせーっちゅうんじゃ！
