----------------------------------------------------------------------
map_chip_id_replacer (2012/12/30)
----------------------------------------------------------------------

▼概要
  ・整理前マップチップ画像及び、整理後マップチップ画像を入力し、
  　整理前マップチップ画像に対応したマップデータのマップチップＩＤを、
  　整理後マップチップ画像に対応したマップチップＩＤで書き換える

▼使い方
  1. mapdataフォルダにマップデータを入れる
  2. map_chip_id_replacer.exeと同じフォルダに、
     整理前マップチップ画像（名前：map_chip.bmp)を置く
  3. map_chip_id_replacer.exeと同じフォルダに、
  　 整理後マップチップ画像（名前：map_chip_new.bmp)を置く
  4. map_chip_id_replacer.exeをダブルクリックする
  5. mapdataフォルダに入れたマップデータが整理後マップチップ画像に対応した
  　 チップＩＤで書き換えられている
  6. 本体に、mapdataフォルダのマップデータ及び、
  　 整理後マップチップ画像をコピー
