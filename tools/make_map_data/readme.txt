make_map_data 2.0 (2010/05/30)
----------------------------------------------------------------------

▼概要
  ・マップ画像及びマップチップ画像を読み込み、マップデータを出力する

▼プログラム製作者
  ・BETA

▼機能
  ・マップ画像及びマップチップ画像を読み込み、FF7FC向けマップエディター
    「FF7FC MapEditor」で読込み可能なマップデータを出力する

▼使用例
  ・本ツールはコマンドラインツールのためコマンドプロンプトで実行する。
    (例 Cドライブ直下に本ツールを解凍した場合の手順)

    1. C:\make_map_data\の下に、マップ画像及びマップチップ画像をコピーする

       ※!!注意!! 本ツールに入力可能な画像形式は24bitのビットマップのみ

    2. コマンドプロンプトを立ち上げ、C:\make_map_data\に移動する

       cd C:\make_map_data\

    3. 以下のように入力し、プログラムを実行する。
       -m の後にマップ画像の名前
       -c の後にマップチップの名前
       -o の後に出力するマップデータのID
       を入力する。

       (例)
       マップ画像の名前がmap.bmp, マップチップ画像の名前がmap_chip.bmpの場合

       C:\make_map_data>make_map_data.exe -m map.bmp -c map_chip.bmp -o 2

    4. 同じフォルダにmapdata_002というような名前のマップデータが出力される。
       02の部分は入力したマップID

※注意
  ・入力するマップ画像の幅は256の倍数のみです
  ・入力するマップ画像の高さは240の倍数のみです
  ・マップチップ画像に含まれていないチップ画像をマップ画像で使用している
    場合はマップチップID:16のチップ画像が出力されます。
    その場合はマップチップ画像を拡充してください。
  ・本ツールで出力される情報は背景扱いとなります
  ・当たり判定情報やワープ情報には全て0が格納されます。

以上。
